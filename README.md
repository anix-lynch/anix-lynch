### ☕ [Coffeeverse: Multi-Agent AI Automation System](https://github.com/anix-lynch/coffeeverse)
**Autonomous AI agents that actually work** - Not just chat, but real workflow automation

**The Problem:** Marketing teams spend 4+ hours weekly creating competitor analysis reports—manually scraping websites, comparing data, writing summaries

**The Solution:** Autonomous multi-agent system that orchestrates specialized AI workers to complete complex workflows

**Architecture:**
- 🔬 **Research Agent:** Web scraping + multimodal doc analysis (Qwen2-VL for images, Llama-3.2-Vision for PDFs)
- 📊 **Data Agent:** SQL queries + RAG retrieval from internal databases
- ✍️ **Writer Agent:** Natural language report generation with formatting
- 🎯 **Reviewer Agent:** Quality checks + fact verification before final delivery

**Key Achievements:**
- 🤖 **Full autonomous workflow** using LangGraph state machines (no hardcoded logic)
- 🧠 **Multimodal RAG pipeline** combining text + image embeddings (FAISS/Pinecone)
- ⚡ **80% time reduction** (4 hours → 30 minutes for weekly reports)
- 🔧 **Tool-calling agents** that autonomously select and execute actions (Google Search, SQL, APIs, calculators)
- 🎛️ **Real-time monitoring** via Streamlit dashboard (watch agents collaborate live)

**Tech Stack:** LangGraph • LangChain • Qwen2-VL • Llama-3.2-Vision • FAISS • Pinecone • FastAPI • Streamlit • Docker

**Business Impact:** Marketing teams now focus on strategy instead of data collection. Competitor intelligence delivered consistently every Monday morning.
